מה נעשה?

נספור את הניקוד של השחק ונציג אותו על גבי המסך

מתי?

תוכלו לענות על שלב זה אחרי שלמדתם על פקודת format


פקודת format

פקודת format מאפשרת לנו לשלב טקסט ומשתנים ביחד. באמצעות הפקודה נוכל להכניס משתנים במקומות מסוימים בתוך המחרוזת. לחצו על כפתור run כדי לראות מה התוצאה שתודפס.

Main Result
name = "Dor"
age = 12
string = "My name is {} and I am {} years old".format(name, age)
print(string)
